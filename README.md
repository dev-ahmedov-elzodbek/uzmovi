# 🎬 Kino Katalogi

## Ishga tushirish (VS Code)

```bash
npm install
npm run dev
```

Brauzerda `http://localhost:5173` oching.

## Stack
- **React 18** + **Vite**
- **React Router DOM** — routing
- **Zustand** — global state
- **Tailwind CSS v3** — styling
- **shadcn/ui** komponentlari (Button, Badge, Input, Card, Separator, Toast, Dialog)
- **Radix UI** — accessible primitives
- **Lucide React** — icons

## Funksiyalar
- ✅ Film katalogi (grid)
- ✅ Qidiruv + janr filtri + saralash
- ✅ Detail sahifasi (animatsiyali IMDb ring)
- ✅ Sevimlilar (Zustand)
- ✅ Toast xabarnomalar
- ✅ Ko'zga qulay teal rang sxemasi
